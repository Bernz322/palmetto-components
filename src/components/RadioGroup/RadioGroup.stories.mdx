# RadioGroup

Use a RadioGroup when a user is required to select one of five or less options. It is ideal for this scenario because the options are displayed without having to interact. If there are more than five options, use the [SelectInput](?path=/docs/components-form-inputs-selectinput--all) instead.

      {

        value: 'vanilla',
        label: 'Vanilla',
      }];
      return (
        <RadioGroup
          name="noTitleOrDescription"
          value={value}
          onChange={event => setValue(event.target.value)}
          options={options}
        />
      );
    }}
